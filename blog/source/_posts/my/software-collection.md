
# Win平台

	---------- 资源管理 ----------
	Divvy，分屏
	Launchy，应用启动工具
	Dropbox，
	Evernote，
	Everything，
	Git
	GitShell

	----------编辑器----------
	notepad++，可扩展丰富的插件，windows平台。
	sublime
	typora，markdown编辑器，即时可见，可自动生成表格，跨平台。

	xmind，思维导图工具，免费，跨平台。
	MindManager，思维导图工具，界面和图形比xmind好看。收费。
	Intellij
	Eclipse
	
	----------办公自动化----------
	everything，全文搜索文件。
	1Password
